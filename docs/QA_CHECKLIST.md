## P0 必須シナリオ

### 1. 初回オンボーディング完走
- [ ] Screen Time 許可ダイアログが表示される
- [ ] 許可拒否時、設定への誘導が表示され Continue が無効
- [ ] 許可後、アプリ選択 → 開始時刻 → タスク設定まで完走
- [ ] オンボーディング完了後、Post-onboarding Paywall が表示（Free ユーザーのみ）

### 2. キル状態でのブロック発火
- [ ] 開始時刻を 2 分後に設定
- [ ] MornDash を完全に kill（App Switcher からスワイプ）
- [ ] 開始時刻到達後、ブロック対象アプリに Apple シールドが表示される
- [ ] MornDashMonitor 拡張が独立動作している

### 3. タスク全完了 → シールド解除
- [ ] ブロック中に MornDash を開く → BlockingView 表示
- [ ] 全タスクを完了 → シールド解除
- [ ] ブロック対象アプリが正常に起動できる
- [ ] 完了演出（RoutineCompleteCelebration）が表示

### 4. Give up フロー
- [ ] BlockingView の "give up" リンク → EmergencyUnlockView
- [ ] スライド確認で give up 実行
- [ ] 当日は idle 状態（BlockingView が出ない）
- [ ] シールドが解除される

### 5. 23:59 境界
- [ ] タスク未完了のまま 23:59 を跨ぐ
- [ ] OS 上のシールドが解除される（Monitor intervalDidEnd）
- [ ] MornDash UI が idle になる（BlockingView が出ない）
- [ ] アプリを開いても再ブロックされない

### 6. 翌日リセット
- [ ] 日付変更後、タスクが未完了状態にリセット
- [ ] 開始時刻に再ブロック発火
- [ ] ストリークが正しく更新（完了日のみカウント）

### 7. Screen Time 許可の再設定
- [ ] 設定 → Screen Time → MornDash の許可を OFF
- [ ] MornDash を開き、ブロックが効かないことを確認
- [ ] 設定から許可を ON に戻し、正常動作を確認

### 8. タスク全削除ガード
- [ ] Tasks タブで最後の1タスクを削除しようとする
- [ ] アラート「タスクは1つ以上必要」が表示され、削除されない

### 9. Pro 購入 / 復元
- [ ] Sandbox Apple ID で Paywall から購入
- [ ] Pro 機能（無制限アプリ、曜日別スケジュール）が有効
- [ ] 復元ボタンで以前の購入が復元される

### 10. 無料枠制限
- [ ] ブロック対象: 2 アプリまで
- [ ] タスク: 3 つまで
- [ ] タイマータスク: 1 つまで
- [ ] カテゴリブロック: 不可（Pro 誘導）
- [ ] Stats 履歴: 7 日分のみ（contribution graph 1 週）
